zetastitcher.fuser.virtual_fused_volume module¶

API to query an arbitrary region in the stitched volume.

class zetastitcher.fuser.virtual_fused_volume.VirtualFusedVolume(file_or_matrix)¶

Bases: object

An API to query arbitrary regions in the stitched volume.

Example usage:

>>> vfv = VirtualFusedVolume('stitch.yml')
>>> vfv.shape
(208, 3, 4533, 2487)

Axis order is ZCYX.

>>> subvolume = vfv[40, ..., 1000:1500, 2000:2400]

Attributes

overlay_debug_enabled Whether to overlay debug information (tile edges and numbers).
shape Shape of the whole stitched volume.
